# Automotive Supplier Quality Dashboard (Power BI)
KPI dashboard analyzing PPAP requests and 8D corrective actions 
across Tier 1 automotive suppliers. Built on simulated, 
confidentiality-safe data.

## Metrics
- Total PPAPs, Average Cycle Days, 8D Closure %
- Defect Pareto (80/20 prioritization)
- Monthly cycle-time trend with trendline
- Supplier scorecard (PPAP volume, 8D closure, VDA 6.3 score)

## Tools
Power BI Desktop · DAX · data modeling (star schema, supplier_id)
